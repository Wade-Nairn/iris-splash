Overview

IRIS Coordination is a desktop application that you must install once per computer, after which you can just run it from the Windows Start Menu.

IRIS is a fire resource coordination and management system featuring a friendly and intuitive windows user interface. IRIS is designed to be used for the day-to-day management of fire and prescribed burn activities involving resources from multiple agencies. It now incorporates the ability for trained users to enter times and allowances for standby and overtime work undertaken; to assist in converting those to payments; and tracking their path through payroll.

IRIS’ predominant use is via a connection through the Department’s network, which permits access to a centralised database – each user can then see the work being done by every other user.
